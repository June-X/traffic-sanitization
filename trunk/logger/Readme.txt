1, Click install.bat to install the http traffic logger and reboot your Windows. It includes a driver (tdifw_drv.sys) and a Windows service (tdifw.exe). uninstall.bat is used to uninstall the http traffic logger.

2, Click uninstall.bat to uninstall the logger.

3, A batchfile executes when Windows starts and the key/mouse logger pops up. Clike OK.

4, All the http and key logs are in C:/Windows/System32/LogFiles/tdifw

5, Tested on US English Windows. If you use other languages, please copy start_logger.bat to your startup directory.
